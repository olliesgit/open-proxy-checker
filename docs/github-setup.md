# GitHub Repository Setup

## About Description

Set the repository description on GitHub to:

```
Local CLI and web UI for collecting, deduplicating, validating, and exporting working proxy lists.
```

## Topics

Add these topics to the repository on GitHub:

- proxy-checker
- proxy-validator
- cli
- nodejs
- web-ui
- open-source
- developer-tools
- proxy-testing
- security-tools

## How to set

1. Go to https://github.com/olliesgit/open-proxy-checker
2. Click the "Settings" tab
3. Scroll to "About" section (under "General")
4. Paste the description above into the "Description" field
5. Type each topic tag into the "Topics" field and press Enter
6. Click "Save changes"
