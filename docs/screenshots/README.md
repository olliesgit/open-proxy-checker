# Screenshots

Screenshots are planned for the following views:

- `cli-help.png` -- CLI `--help` output
- `cli-scan.png` -- CLI proxy validation run in progress
- `webui-home.png` -- Web UI homepage with controls and sources panel
- `webui-results.png` -- Web UI with scan results, latency bars, and export

To add screenshots:

1. Run the CLI or start the web server
2. Capture the relevant view
3. Save as PNG in this directory
4. Reference the screenshots in the README
