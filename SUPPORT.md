# Support

## Issues

Report bugs and request features via [GitHub Issues](https://github.com/olliesgit/open-proxy-checker/issues).

Before opening a new issue, please:

- Search existing issues to avoid duplicates
- Use the provided issue templates
- Include your Node.js version and operating system
- Include steps to reproduce for bug reports

## Discussions

For questions, ideas, or general discussion, use [GitHub Discussions](https://github.com/olliesgit/open-proxy-checker/discussions) if enabled.

## Security

Report security vulnerabilities per [SECURITY.md](SECURITY.md). Do not open public issues for security concerns.
